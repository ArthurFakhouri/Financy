## 🈸 Project
The project is a desktop website for organizing finances, with transaction and category management.

<b>Features:</b>
- [ ✅ ] The user can create an account and log in
- [ ✅ ] The user can only view and manage the transactions and categories they created
- [ ✅ ] It should be possible to create a transaction
- [ ✅ ] It should be possible to delete a transaction
- [ ✅ ] It should be possible to edit a transaction
- [ ✅ ] It should be possible to list all transactions
- [ ✅ ] It should be possible to create a category
- [ ✅ ] It should be possible to delete a category
- [ ✅ ] It should be possible to edit a category
- [ ✅ ] It should be possible to list all categories

## 🔧 How to run
### Backend
Go to the folder where you cloned the project, open the terminal and run the commands to run the server:
- 01 - ```cd backend```
- 02 - ```npm i```
- 03 - Configure the environment variables in a .env file (Follow the example in the .env.example file)
- 04 - ```npm run dev```

### Frontend
Go to the folder where you cloned the project, open the terminal and run the commands to run the web:
- 01 - ```cd frontend```
- 02 - ```npm i```
- 03 - Configure the environment variables in a .env file (Follow the example in the .env.example file)
- 04 - ```npm run dev```
